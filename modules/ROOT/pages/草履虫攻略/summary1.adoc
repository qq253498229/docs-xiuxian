= 练气期总结

== 画符准备

* 画符前先学两个技能：【云气汇流】、【通符书】
* 画一张文昌符，保证心境在90以上，
* 白素贞去藏经阁录取符修功法拿参悟
* 杨树去藏经阁录取零散秘籍
* 白素贞修炼到初符(结丹后期)境界，选择“碧灵法”，使用mod自动画符，绘制第一张本命符【伪通天符】
* 符纸开局送10张，用朱砂和草药可以在制符桌生产

== 16天太一石闸

* 获取前期最重要的【千里神视符】与【玄空洞秘闻】
** 玄空洞秘闻必须无天谴的小人去，体修清理地图已经有不少天谴了，派白素贞去
* 玄空洞能拿到【天通洞神符】（能大量提升心境）和【天机化运法】
** 【天机化运法】录入藏经阁
* 太一石闸任务后可以重建势力点，派垃圾小人去，后期换成化形猫妖

== 体修突破

* 声望高了袭击来的也越来越频繁，体修取消吐纳权利冲击脱胎期

== 紫霄宫

* 开启紫霄宫，派杨树进入，获取前期最重要的资源，【神珀符】与【启智丹】。

* 白素贞在【神珀通天符】和【天机化运法】协助下，需要205道行可以100%大衍神算成功
