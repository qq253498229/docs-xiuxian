= 总结

== 开局设置

* 牛妖带队
** 1、化形天劫不能太近
** 2、版筑、巧匠至少一个热爱，且等级上限至少20
** 3、选择体修功法、热心救人、药庐遗物
* 4、带杨树（后期掌门，可换其他）、白素贞（不可换）、鱼头君（前期打手，可换）
* 如果随机到乌龟建议重开

== 家里

* 采集龙涎果给外门吃，吃一个半天不用吃饭睡觉
* 采集银杏果给体修吞噬
* 外门除草，搞2个灵初露给体修吃，补充800精元
** 再搞两个灵初露放烂掉，有掌门喜欢垃圾
* 大地图挨个门派拜山，每个门派送一桶水

== 体修淬体

* 优先使用生命本源
* 其次使用生命精华
* 开局先淬炼左右前腿（如果是人就是左右手）的4块肌肉
* 词条只点【百炼血肉】、【侵蚀之血】、【元祖化】（不到一天就能到二重，然后就可以出去打野了）

== 第一桶金-打野

* 第一张打野图是【铜陵山】
* 铜陵山里一共两只妖兽
* 屠宰获取【妖灵血】【妖兽皮】
** 妖灵血是火属性，再家里浅水区规划一块仓库专门放火属性物品
* 妖灵血和妖兽皮卖掉，攒400买极天宫通行证

== 解决温饱

* 收获莲藕、梨以及龙涎草，作为开局的粮食
* 砍树，开始只砍灵木（搭建聚灵阵）
* 攻击性动物让体修屠宰掉（虎、狼、熊、蛇、猫、牛、猪）
* 除草获得灵初露，放在置物台上做成蓝图，之后会自动收获灵初露

== 搭建聚灵阵

image::聚灵阵.png[]

* 如果开局没从箱子开除布料，可以去小凉山拿棉花回来自己织布

== 获取符修功法

* 极天宫掌门送心仪之物（这张图是垃圾），400灵石买通行证
* 找到何足道，用mod一键满好感，送礼符纸，返还【太元五符云箓】（即符修功法）
* 走之前拿点饭，挑数量多的拿

== 体修继续打野

* 体修拿到符修功法后继续铜陵山打野，直到买下门派卖的所有猫咪，四只乌龟，一直老虎，买便宜的
** 所有门派不同，但每10天补一次货
** 最便宜的就是两个灰色笼子，这种笼子的动物神识必满，比华丽笼子好很多
* 同时刷声望，配合仙家洞天，冲击二级门派
* 买下所有门派的通行证

== 开启太一遗迹，拿建筑材料

* 白素贞和杨树一起修炼【太元五符云箓】
* 修炼目标改成【调心】
* 换上品质不是糟糕的衣服
* 挑一个符修去月轮山（体修继续打野），隐匿（防止被揍），右下角太一山门，拜山
* 开启【太一遗迹】后，两个内门一起去太一遗迹驻扎。目标：
** 1、塔中的太一秘宝
** 2、香囊
** 3、满地的建筑材料，只拿100+的，多来几次搬空
