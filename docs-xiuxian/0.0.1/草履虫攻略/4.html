1. 聚灵物原理

  1. 修炼时，灵气浓度可以提高修炼效率等。

  2. 聚灵物可以提升地块的聚灵值（缓慢提升到一个数值），而聚灵值决定了平静状态下地块灵气浓度上限（目前理解但这个程度就行了）。

  3. 不同聚灵物有不同的聚灵值和聚灵距离限制。而本游戏的两点间距离满足勾股定理。

  4. 聚灵范围在实际距离计算中需要-1。比如琅琊果的聚灵范围是2，但实际上它只能影响本格与周围上下左右四个格子的聚灵值。

  5. 我们要做的就是通过合理摆放聚灵物在小人打坐处获取最高聚灵值，以拉高灵气浓度。

  6. 而灵木聚灵范围是4，经过勾股定理计算，我们最终摆出的聚灵阵就是5*5+4。

  7. 至于外围的4个卫星灵木也是用于爆灵使用。

  8. “爆灵”对聚灵阵的高阶利用方法，后续用到会详细讲解理论与使用方法
