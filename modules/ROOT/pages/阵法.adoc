.上古阵法需求

|===
|阵图名称|所属门派|需求

|诛仙剑阵	|七杀魔宫
|全体要求：元神期，御器25级以上，道行350以上。 +
阵枢：- +
阵眼：每个阵眼佩戴对应秘宝，分别为诛仙剑、戮仙剑、陷仙剑、绝仙剑

|北斗真武戮魔大阵	|七仟坞
|全体要求：元神期，御器25级。 +
阵枢：学习天权诀，佩戴秘宝-天权剑 +
阵眼：按位置需求，学习北斗洞心劫法中的对应技能（诀），佩戴对应名称的秘宝。

|天地五方不灭阵	|玄一道
|全体要求：元神期以上，最大灵气64W +
阵枢：在世真仙 +
阵眼：每个阵眼佩戴对应秘宝，分别为青色云界旗、青莲宝色旗、真武皂色旗、离地焰光旗、戊己杏黄旗。

|诸天六御行法仪轨	|昆仑宫
|全体要求：雅艺、处世20级。 +
阵枢：男性， +
阵眼：5人中有1个女性，4个男性。

|九御赋龙形阵	|百蛮山
|全体要求：元神期 +
阵枢：术法25，佩戴蛟龙的逆鳞，参悟龙魂真灵法。 +
阵眼：按对应位置，2人佩戴蛟龙的天角，4人佩戴蛟龙的罗刹爪，1人佩戴蛟龙的玄鳞，1人佩戴蛟龙的崩天筋

|真凰羽剑阵	|极天宫
|全体要求：元神期至少3人，金丹期以上6人 +
阵枢：元神期，御器25，佩戴凶凤真焱玉，参悟凤魂真灵法。 +
阵眼：元神期2人，分别携带凶凤的烬灭羽、凶凤的蔽天羽。其余6人分别需要携带2个凶凤的空坚骨，4个凶凤的刚羽。

|阴阳二行神阵	|紫霄宫
|全体要求：元神期 +
阵枢：混沌的龙玉、道行360以上，参悟阴阳真灵法。 +
阵眼：一男一女，都携带阴阳之结晶。

|三阴三绝阵	|合欢宗
|全体要求：女性，元神期。 +
阵枢：- +
阵眼：-

|八门玄金天劫剑阵	|青莲剑宗
|全体要求：元神期，7个阵眼中至少有一男一女。 +
阵枢：梧桐神木，御器20级。 +
阵眼：梧桐神木1个，星髓1个，天劫之息2个。

|先天甲乙木神阵	|丹霞洞天
