守家大哥培养

  • 初始功法选三阳火，实际上选哪个都差不多

  • 春天属木，适合火功法结丹，并且聚灵阵用灵木做，初期就能搞出来

  • 缺点是最佳结丹时间离开局比较近，萌新主修火的话不太容易赶上最佳结丹时间，晚个几天也不很关键，反正这个阶段小人谁都打不过，重点是有内门小人就可以舔地图收集资源了

  • 内门小人先不急立刻出去舔图，先画几张轻身符给外门，提高干活效率

  • 根据模板临摹几张伪造的分影遁空符和神行符，提高内门舔图效率，给历练快的那个再画个伪造的千里神视符，可以稍微提高一点点开地图的效率

  • 内门小人一个去开南屏山拿弓和肉，一个去开太一遗迹，拿建筑材料，然后把老家门口几个地点的村子都舔一遍，应该会搞到100-200个石髓，用石髓买极天宫通行权

  • 外门小人开局任务是搭聚灵阵，开农田，有建筑材料之后建房屋和工作场所，找个猎户评分高的拿弓箭逐步清野兽弄食物

然后

  • 去天极舔何足道拿符修功法，以及后面特殊事件把五种无上大法拿到，

  • 首发工具人结丹，月轮山猎户事件拿符法和丹方，到这一步很多攻略都有讲，就不再重复啰嗦，重点是到这一步往后怎么走

这时你要考虑

  • 前面几个内门工具人都是弟中弟，功法不全，聚灵阵只有个灵木聚灵阵，

  • 五本无上大法再加个符修功法，就这六本功法都没有足够的参悟去学，一整个门派都没个像样的法宝，

  • 总之就是缺书缺人缺钱缺参悟缺材料什么都缺，千头万绪不知道干啥

问题总要一个个解决

  • 先说聚灵阵，除了木系聚灵阵，最容易成型的是啥？土系聚灵阵。

  • 金系需要练个土功法的小人反复历练拿星髓，水系需要反复历练搓寒精锭，火系也是反复历练搓火锭，

  • 而土系只需要你带一坨屎，潜龙渊直接拿9个天龙砂，东北有个村子也有一个天龙砂，玄一道（记不太清，如果记错了轻喷）一次可以偷一堆天柱石，如果降低要求用玉髓更是跑一趟就能集齐聚灵阵物资。

  • 初期哪个聚灵阵效果最好？其他聚灵阵灵植75聚灵，范围还只有2，其他聚灵物都是35，唯有我天龙砂聚灵范围聚灵值双料称霸

  • 初期哪个聚灵阵最有用？符修两仪两位大哥表示我不看五行，只看聚灵值，北斗和七杀两位小哥表示我就喜欢坐一堆天龙屎中间尽情呼吸，其他兄弟纷纷表示我就是个工具人

  • 综上所述，除了起手的灵木聚灵阵，土系天龙屎聚灵阵是最重要的聚灵阵

那么如何最快的速度弄到这玩意呢？

  • 拿到符修功法后，找个小人学符修进入内门，啥技能都不要学，闲下来就练功突破攒参悟，

  • 优先把分影遁空和千里神视两种符学了，

  • 然后带符历练紫霄宫拿到一堆神珀符，

  • 再装上神珀符画的分影遁空和千里神视去历练开潜龙渊地图，

  • 拿屎速出土系聚灵阵
