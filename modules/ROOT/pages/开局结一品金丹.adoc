= 一品金丹条件

1. 拿符修和武当两仪功法
2. 【地母灵液1】个
3. 接单需要心境95以上，画通天符加5维
4. 聚灵阵
5. 天气（火系接单用驱云诀）

== 开局地母灵液

合欢岛或者灵性灵植蕴养灵初露（煞性蕴养就是邪脉血泉）

* 种灵植需要五系灵根(灵初露放在置物台上避免腐烂，用蓝图建造自动拾取)
* 赭黄精：高机缘的火系小人去炼丹峰历练（白素贞）
* 木枯藤：龙虎山玄一道仓库里多
* 琅琊果：极天宫左边偷

== 极天宫获取符修功法

* 购买通行证：40石髓
* 何足道满好感，送心仪之物（符纸）就会返还【太元五符云箓】就是符修功法
* 何足道满好感，问一下绝学，会返还秘籍【足道神视】
* 注意功法会掉在地上，一定要捡起来

== 画符

* 灵血符：妖灵血：去野外地图杀妖兽
* 神珀符：宁珀：灵植蕴养尸体
* 金丝灵木到玄一道购买(顺便买个【糊泥刀】，加建设速度非常多)

重要符咒

* 遮天符：减道行，学习的时候省参悟
* 借天符：换道统
