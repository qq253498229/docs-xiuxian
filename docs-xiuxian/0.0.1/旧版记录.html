  • 孙亚龙带队，陆夫人，申申如也，鱼头君

  • 孙亚龙是木轮回开局，建设弟子带初始资源

  • 陆夫人五维高，丹器高，负责炼丹

  • 申申如也手艺人，可以在外门做东西

  • 鱼头君专门跑图和偷东西。

  • 常规开局，快速积累门派资源，然后摸索体修和妖修。

家里开局流程：

  • 鱼头君进直接吃筑基丹进内门，画12张轻身符，每人一张，

  • 鱼头君初始20级斗法，进内门御器等级高（随机），高御器适合跑图拿资源。

  • 每个门派拜山一次，送最便宜的一桶水，之后就能驻扎、历练、交易了

  • 孙亚龙带队(带建筑弟子天赋)多给200木头，200铁，200石砖，40火铜锭，40寒晶锭，可以快速建家

  • 在泥地或者湿地上建造木质手工桌和砍柴处，中间的一个空造寒晶灯笼，两个风水建筑大吉直接仙家洞天。

  • 仙家洞天送300声望，早点500声望解锁2级门派。500声望后再考虑压声望，人多才能更快速的发展。

  • 砍灵木，劈灵木料做简易聚灵阵。筑工要取消了。调成苦力优先搬材料，不会建造，搬完材料，取消，全部禁用，完成后再把筑工选上

  • 招来的小人：属性好的帮忙干活，属性差杀掉，埋人加20声望和20正义。早点到500声望，需要埋10个人

  • 朱果温度高，到家后可以找一个小人装备到身上，这样就不会着火了

清理一块地方把朱果种下去(没有超级修炼堂了，不着急种了)，蕴养小动物尸体，可以获得宁珀和幽珀，高级符纸需要宁珀

鱼头君前期流程：

  • 先去去小凉山驻扎，拿石髓、棉花和2个矿稿，用40石髓换蜀山通行证，

  • 然后去蜀山拿饭，1个珍馐3个普通，属性好的外门可以邀请到门派，帮忙干活，

  • 然后把外门小人都邀请到家里，之后偷左下的法宝，然后绕到下面，沿着地图边缘走（这样不容易被发现）到右上偷朱果

  • （白色物品随便拿，其他物品需要隐匿，周围不能有人（外门不影响））

  • 继续到月轮山，拿6样东西，把村农都邀请回门派，月轮山地图最右下角山门可以拜一下，开启太一遗迹

  • 继续到太一遗迹，拿点灵石砖和铁锭，捡数量多的拿，记得拿香囊增加修炼的心境

  • 继续到南坪村邀请村民，顺便偷东西

  • 聚灵阵需要5天聚灵时间，鱼头君调心，然后开始修炼

  • 学了御灵术之后，用寒晶锭造1*3的冷冻房，把神秘者冻死，御灵术把尸体做傀儡，元神傀儡搬东西非常厉害，这也是木轮回开局的优势

接下来：

  • 送掌门心仪之物，2个心仪之物可以到友好关系

  • 友好关系丹霞山可以买赤参，极天宫能买紫芝

  • 孙亚龙吃2赤参2紫芝进内门，鱼头君开始修炼，孙亚龙接替鱼头君去偷东西

  • 然后去太一遗迹偷东西，太一遗迹里的塔，可以拿到太一遗物

  • 2个内门轮流守家

  • 邀请的村民，属性高的招过来干活

  • 如果农民多，可以把地图里的草药，银杏等资源收集一下，都能卖钱

  • 太一遗迹拿的灵石砖和铁锭，做灵石枪卖钱

  • 埋人加声望，目标500声望升级门派，继续招人，

  • 人多发展效率高，500声望会来练气期敌人，留内门守家

  • 鱼头君随便结个金丹，结丹后，需要突破新生期，再出去历练

  • 地图上进攻性的动物都杀了，肉全做肉干

  • 由于布施需要小麦，小麦要多种一些

  • 龙虎山买金丝灵木和宁珀，做高级符纸

第16日：

  • 出现太一石匣剧情，学习秘籍

  • 用高级符纸画神行符和千里神视符，加快开图效率

  • 岭南重建代理（代理工具人选魅力高的小人，布施任务魅力高加的人口多，最好战斗也高点，因为有时候会触发战斗选项）

  • 鱼头君带神行符到丹霞山借东西，属性好的外门邀请到门派

  • 等周围没有元神大佬，拿20个筑基丹和门口的法宝

  • 鱼头君继续到昆仑宫观摩炼法坛，左边赭黄精可以拿，往上面走，下面人比较多

  • 继续到丹霞山拿灵晶，炼法坛需要灵晶

  • 鱼头君带千里神视符，优先探索九华山，

  • 九华山出现凤栖崖后，拿4个凤羽就能烧大能拿符、法宝、装备了

  • 紫霄宗邀请门派大佬（准备烧），需要遮天符，借天符，通天符和洞神符

  • 跑一趟极天宫邀请何足道（准备烧）

  • 潜龙渊拿天龙砂（需要提前准备粪便，进去后吃了然后摸门）

  • 栖霞洞天观摩小丹炉，小丹炉很好用（没五行，占地方小）

烧大能：

  • 准备：火羽装备到身上，家里内门可以学求雨术（最好有木雨剑）防止火羽烧大能的时候着火

  • 大地图买点极乐丹，美轮美奂的大吉房间，灵木料墙，火铜床，2个寒晶锭置物台，置物台上放灵木

  • 开烧：邀请的大佬直接用墙围住，隔墙扔火羽，在人物周围造墙卡住位置，把火羽卡到房间里，再把墙取消

  • 大能烧晕后，扒装备，然后换到大吉房间，放能加心情的法术(比如放5次化伤术)

  • 烧出来：借天符，遮天符，通天符和洞神符比较关键的符咒

家里：

  • 做肉干和灵石枪

  • 陆夫人吃赤参，紫芝或者熊胆，进内门学土葵花（太一石匣送的秘籍）

  • 把功法录入藏经阁（录夫人）

  • 金灵根-琅琊神木，灵植蕴养地母灵液，灵性可以出灵髓脂

  • 第21日,商人来了，炮仗买一个炸小妖,家里的肉干，太一遗迹搬的资源多换点灵石，破烂都卖了，钱多了就可以无限做灵石弓刷钱了

  • 等内门从潜龙渊拿到天龙砂了，做天龙砂聚灵阵，土生金，适合练金北斗小人守家

  • 做点妖兽皮薄衣裳和薄裤子

  • 聚灵阵造好后，陆夫人结一品金丹，通天符加五维，出筑基丹临时加属性，陆夫人尽量结一个高品金丹，负责幽淬和4维淬体，之后还会负责炼丹

  • 小乾坤界完成，早点儿建造，非常好用

  • 陆夫人带借天符，大衍神算

  • 派小人带通天符加五维，去道统现世的地方历练拿齐功法

  • 天龙砂聚灵阵非常好用，金北斗结丹，专门守家

跑腿工具人无限历练拿材料：

  • 机缘高（5以上），去岭南拿参悟，带上遮天符（减道行加点消耗低），把御器（增加移动速度）学满

  • 然后带神行符（移动速度+历练速度）+飞影遁空符（长途御剑的飞行速度，也就是历练速度）

  • 练气期就够了，因为练气期可以出现和弟子交流的选项，可以拿双倍物品

  • 五莲山：紫芝

  • 丹霞山：赤参

金丹后修炼：

  • 陆夫人土系，去餐霞洞历练刷修为（偷天诀也是）

  • 水太和功法，金丹期，学求雨术，然后去五莲山的木白雨林拿木雨剑

神修功法：

  • 岭南主殿升级到中级可以获得一本

  • 开启古旧石匣有几率获得一本

  • 古代密卷有几率开出一本

  • 还有一个剧情的，由于时间太晚就不考虑了

体修功法：

  • 114天任务送一本：太古形意六合真体

  • 大漠篇可以提前开启万妖殿，如果没有则在114天事件开启

  • 万妖殿可以偷一本体修的功法

  • 万妖殿友好1200之后拜访掌门会送一本体修的功法

  • 万妖殿一个长老会一本体修功法，可以打死之后用搜魂大法（如果没有的话在160天有个任务可以出）搜出来

阵法书

  • 铜陵山：阵法书

  • 天池：阵法书

  • 百蛮山-万兽窟：阵法书

  • 白木雨林：阵法书（吃蘑菇之后观察5个五行灵根，之后去白木雨林触摸石碑）

  • 小人身上带着冬虫、夏草到天池驻扎进地图可以拿到阵法书。进地图后吃掉冬虫和夏草，然后触摸中间石碑。（里面还有个野生的长生泉）

  • 冬虫、夏草可以在大地图交易，或者去虫谷，冬天出冬虫，夏天出夏草。

聚灵阵：

  • 聚灵阵最外围用灵木

  • 天龙砂聚灵值很高，不需要练法坛，其它属性的优先练法坛（600灵气去昆仑宫观摩），等有了龙凤材料再换掉练法坛

  • 天龙砂(潜龙渊吃粑粑拿)

  • 天柱石（丹霞山仓库偷）

  • 金系：九转金丹+星髓

  • 水系：五色金莲+玄牝珠

  • 木系：木枯藤+梧桐神木

  • 火系：朱果+火精锭

  • 土系：造化神丹+天龙砂+天柱石

  • 土系不需要练法坛，天龙砂灵气高

进攻（骨骼）：

  • 紫色：燃炽之骸（威力480%）+吞灵之骨（穿透120%）

  • 蓝色：玄煞之骨（穿透基础4.5%）+激生之骨（威力360%）

  • 绿色：金灵化（穿透基础1.28%）+锐化骨骼

  • 白色、橙色：百炼骨骼+元祖化

进攻（肌肉）：

  • 紫色：渴血之肌（威力240%+命中）

  • 蓝色：灭活血肉（威力360%+穿透）+侵蚀之血（威力270%）+凋零之血（穿透45%）+血肉增生

  • 绿色：金灵化（穿透基础1.28%）

  • 白色、橙色：百炼骨骼+元祖化

防御：

  • 紫色：空化之力（闪避2%）+潜默冥息（回复16）+先天返照（回复基础8）+太阳之息（狂怒30%）

  • 白色、橙色：百炼骨骼+元祖化
