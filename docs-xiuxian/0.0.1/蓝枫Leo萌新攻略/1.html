2. 开局设置

  • 地图尺寸大

  • 真仙模式(有mod可以存档)，下面几个不增加难度的萌新也可以勾选：

    • 天涯沦落

    • 一视同仁

    • 丹道失传

    • 腐败之息

  • 地图环境设置：奇异灵兽-蜚(赌一手玄牝珠)，古代遗物-蕴灵树(增加修炼速度)，或地母石碑(批量获取元神妖兽产物)

  • 地图种子：壹壹零捌捌柒零叁肆陆(1108870346)

3. 开局队伍

  • 1号位牛妖随机

    • 化形天劫不能太近，最好100天以上

    • 版筑、巧匠至少一个热爱，且等级上限至少20

    • 选择炼体传承、热心救人、药庐遗物

  • 2号位杨树/飞羽仙狂（可换）

  • 3号位女蛇妖-白素贞（不可换）

  • 4号位男人-鱼头君（前期打手，可换）

  • 开局有乌龟小人建议重开

4. 开局工作

挑一个五维很低、战斗很高的妖怪，吃两个赤参两个紫芝晋升内门，选体修功法，开宗立派

  • 外门时间规划：全选工作（饿/困急眼了自己会去解决的）

  • 救灾、苦力：所有人都干

  • 筑工：2-3个人（包括搬原料）

  • 膳夫：可以不选，去偷饭

  • 剩下巧匠、衣匠、铁匠、木客、石工、农人、猎户、郎中：每样选一个最高的
