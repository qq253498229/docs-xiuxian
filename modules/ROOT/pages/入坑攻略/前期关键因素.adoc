= 前期关键因素

萌新活不下去的几个因素为：

1. 天气原因，冻死了，热死了；
2. 食物原因，这游戏里小麦生长是有条件的，野生动物也不是无限刷新，等到要断粮了才准备解决体验会非常差；
3. 地图起始妖兽苏醒打不过；
4. 妖兽小人化形劫过不去；
5. 蛟龙屠杀

== 天气原因

1.其中的天气问题解决最容易，开局是春天，要三十天后才会热，可以搓几个提高温度上限的符箓，速成一个元神期内门，打野外地图的妖兽弄点妖兽皮搓衣服即可度过冬天

== 食物问题

2.食物问题，开局5天之内开两块20✕20的农田种上小麦，内门去驻扎南屏村，小人到位之后进去地图，打开目标捡野猪肉和弓，回门派后找个猎户评分高的拿上弓把地图的野兽清了，这些食物能撑到小麦收获，如果肉烂掉或者坏掉，可以反复去村子里捡野猪肉，无限刷新的，去一趟大概400-600野猪肉

== 初始的妖兽问题

3.初始的妖兽问题，开局选地图的时候选个蜚或者鹿蜀，这货比普通的野兽稍微强一点点，用根骨高跑得快的小人拿弓箭攻击他，射两箭等他靠近了就跑开继续攻击，放风筝个几次就能杀死，屠宰后必定出妖灵血，有概率出玄牝珠，如果有玄牝珠就放仓库降温，可以减慢食物腐烂速度，如果没有玄牝珠也没关系，用墙盖个房子把妖兽围起来，在不惊醒妖兽的前提下，围出来的房间越小越好，把妖灵血禁用，让小人带上，丢进房间里，用不了多久妖兽就会热死，丢玄牝珠进去冻死他也是一样。记得盖的房子建个门，否则小人有几率关里面出不来

== 关于妖兽弟子化形劫

4.关于妖兽弟子化形劫，对于萌新来说开局选的小人化形劫最好在80天以上，把小人练到结丹，装备玄甲符，不要带法宝，吃个灵石一般就能渡劫成功，这也就意味着需要练一个符修，符可以去武当紫霄宫进入地图去捡神珀符

== 普通模式300天会出现神秘洞穴

5.普通模式300天会出现神秘洞穴，如果填了会立刻出蛟龙，如果不填一段时间后也会出蛟龙，此时可以盖个房间把他围起来，弄个镇物变大吉就可以镇压住，除非你主动去填洞或者风水被破坏，否则他就一直窝在里面不会攻击你。风水镇物怎么来，开局有内门之后尽早去月轮山驻扎进入地图，左下角找到山门拜山解锁太一遗迹，进入遗迹捡各种砖头木料，第21天商人来卖一部分给商人换灵石，有了灵石疯狂搓灵石砖头，用灵石砖头疯狂搓东西，什么手镯针灸之类都可以，猛搓，搓他千儿八百的总归会出现镇物，找个内门小人学风水鉴定，鉴定出镇物条件，灵石镇物不会要求五行条件，其他条件选个比较容易满足的，尽量不要选厨房条件的，如果非得用厨房条件的要让炉灶远离墙，防止小人来做饭失败把墙烧了，发现了没大事，要是没发现这房间就镇不住龙了
