进一步提升战力

1. 本章概览

之前我们花了14天左右完成了所有前期准备工作。

按本次攻略设定，进去练气期（破局关键期），这一个阶段直接决定我们是否能通过九劫真仙最大的考验，既打败80天来袭的四只元神十二境妖兽。

为了达成这个目标，我们必须在接下来的60天内形成足够的战斗力。故设定修炼目标如下：

  • 1、培养一个能算道统且没匹配几乎所有道统的小人

  • 2、拿齐所有道统，包括武当两仪真经

  • 3、探索出所有地图地点，并拿回相关资源

  • 4、培养2-4名好机缘小人历练寻宝

  • 5、建立高级聚灵阵

  • 6、修建小乾坤界

  • 7、在上述工作的基础上开始培养能战胜80天妖兽的仙修打手

  • 8、为打手炼制所需法宝

  • 9、培养四名高护盾好灵气回复阵法充电宝，预备屠龙

后面为可选：

  • 10、开始培养剑修大哥做寄托套（会讲解重生寄托全流程）

  • 11、同步培养法修大姐做寄托套

  • 12、从零开始养两头牛，预备顶配体修与超级外门

  • 13、有余力的情况下培养一个幽冥神修

  • 14、冲击三级门派，建立修炼堂，并培养一名修炼堂主

  • 15、培养炼丹小人

  • 16、速飞升一名偷天诀仙人

  • 17、准备制作减道行套、修炼套以及外门套

  • 18、集齐御风子游记，挖矿开出三洞玉函

  • 19、买鞭炮炸春节小妖，吃饺子拿春节衣服

2. 画符前准备

  • 初符画符前，需要学习两个技能：云气汇流、通符书

  • 画一张文昌符，保证心境在90以上，白素贞去藏经阁录去符修功法拿参悟。

  • 杨树录去零散秘籍。

  • 学技能都在藏经阁学习

3. 画本命符

  • 白素贞达到初符境界后，开始绘制第一张本命符，伪通天符

  • 左下角选择“碧灵法”，这种画法消耗10倍灵气，同时符咒产生2倍效果

    • 开mod自动画符

4. 培养猫和龟

  • 同步培养买来的猫和龟。

  • 买的动物自带两条完整思绪，也就是说，我们只需要培养一条思绪即可以化形。

  • 猫随意培养尽快化形，龟要多拿几条绿色加护体的，不着急化形

  • 思考界面需要三条相同片段才能思考，但我们可以在有两条的时候去观察特定物品获取一条，来凑够三条，也可以记住需要的片段。

时间来到第16天，太一石闸的任务终于来了，赶紧拿回来，打开后，获取前期最重要的【千里神视符】与【玄空洞】秘闻

    • 注意玄空洞秘闻必须无天谴的小人去，我们的体修开局清理地图野兽已经有不少天谴了。

  • 这里派白素贞去。

  • 天机化运法也录入藏经阁

  • 伴随太一石闸任务的是重建势力点，派一个最差的小人去就行，后期可换成化形猫妖
