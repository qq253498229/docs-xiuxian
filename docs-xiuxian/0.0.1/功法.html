功法

1. 仙修

  • 金-北斗洞心劫法(重回太一)

  • 木-长生六道轮回经(重回太一)

  • 水-太和十六洞天(重回太一)

  • 火-三阳三昧丙丁炼火诀(重回太一)

  • 土-葵花炼神大法(重回太一)

  • 金-七杀剑诀(大衍神算)

  • 木-己寅九冲多宝真解(大衍神算)

  • 火-太上忘情道(大衍神算)

  • 火-九转金丹直指(大衍神算)

  • 土-偷天诀(大衍神算)

  • 玉清仙法(大衍神算)

  • 太元五符云箓(何足道)

  • 玄天真武两仪妙道(悟性+机缘≥16,金丹境界进武当山【金顶】参悟灵石金殿)

2. 神修

  • 九天应元雷声普化玉书(升级岭南代理-秘闻失传之法)

  • 诸天八部成就法(开启古代密卷有一定几率获得)

  • 幽冥十方太乙救苦经(开启古代密卷有一定几率获得，挖掘矿脉的掉落的石匣有几率开出)

3. 体修

  • 太古形意六合真体(第114天，固定秘闻龙崖石刻)

  • 蛮极金身(万妖殿好感度满值后，拜访掌门获得，大漠篇可以提前开启万妖殿)

  • 妖祖圣体(开古代密卷，或拉拢、搜魂万妖殿弟子)

  • 败灭邪体(万妖殿偷。或拉拢、搜魂万妖殿弟子)
