获取符修功法

  • 打开天下面板，挨个拜山每个门派，一桶水就行。

  • 然后给极天宫掌门送上一份垃圾（这图极天宫掌门喜欢垃圾），接着用400灵石换取通行证

    • 之前的第一桶金就用在这，妖兽皮一个值4灵石，妖灵血一瓶值40灵石

  • 进入极天宫找到何足道，一键满好感

  • 送心仪之物(这个种子是符纸)，返还【太元五符云箓】就是符修功法

  • 走之前记得“拿”点饭

    • 等体修到家得过程中可以去 搭建聚灵阵

  • 体修到家之后继续去铜陵山杀妖兽刷材料，直到买下所有门派卖的猫咪以及4只乌龟外加1只老虎，只买便宜的

  • 同时刷声望，配合仙家洞天给的声望，冲击二级门派

  • 还要买下所有门派的通行证

  • 买一只老虎是为了当大哥，也可以用人族
