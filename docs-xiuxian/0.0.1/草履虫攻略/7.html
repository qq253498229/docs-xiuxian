1. 接下来的事

接下来一段时间，我们需要做的主要事情有几点：

  • 1、继续拿道统和古书

  • 2、将除杨树外的，所有内门小人修炼至结丹瓶颈，做好结丹准备

  • 3、建造小乾坤界

  • 4、拿回足够量的地母灵液

  • 5、淬炼出第一批圣品符咒

  • 6、把外门牛的巧匠冲击到20级，制作法宝胚子与全家妖兽皮衣服

先给白素贞补充一点参悟，等偷天诀拿到之后学寿元技能，然后咱们起修炼堂，白素贞暂代掌门与堂主之位：

  • 1、白素贞学习祈天经，抽空去一趟木白雨林，取回木鱼剑（这把剑扔地上就可以召唤小雨，可以抵消恶劣天气）：

  • 2、多宝猫猫到达“万象宫”瓶颈之前，学习“同心诀”拉满处事（一定要到瓶颈之前学习）：

  • 3、继续算道统，取回最后一本无上大法《偷天诀》，并积攒天道神丹：

  • 4、将在各个地图取回的灵根种植下来：
